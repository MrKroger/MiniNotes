============================================================
           CONSOLE ЗАМЕТКИ - ИНСТРУКЦИЯ
============================================================

ОПИСАНИЕ:
Простое консольное приложение на C++ для управления списком 
задач. Поддерживает интерактивное меню, ведение статистики 
и выбор места хранения данных.
Файлы config и notes, при их отсутствии, создаются 
автоматически. 

ОСНОВНОЙ ФУНКЦИОНАЛ:
1. add - Добавление заметки с меткой времени.
2. list - Просмотр всех заметок (статус, текст, дата).
3. done - Отметка заметки как выполненной.
4. undone - Возврат выполненной задачи в список дел.
5. delete - Удаление конкретной заметки.
6. clear - Удаление всех выполненных задач.
7. stats - Автоматический подсчет прогресса в главном меню.

ТЕХНИЧЕСКИЕ ОСОБЕННОСТИ:
- Язык: C++.
- Библиотеки: Только стандартные + системные API Windows.
- Типизация: Строгая статическая.
- Защита: Обработка неверного ввода, пустых строк и 
  ошибок доступа к файловой системе.

ФОРМАТЫ ФАЙЛОВ:
1. База заметок (notes.txt):
   Формат: [статус]|[текст] ([дата и время])
   Пример: 1|Купить молоко (27.05.2024 14:00)

2. Конфигурация (config.txt):
   - Строка 1: Полный путь к базе заметок.
   - Строка 2: Общее количество задач.
   - Строка 3: Количество выполненных задач.

УПРАВЛЕНИЕ:
- Стрелки [Вверх]/[Вниз]: Навигация по меню.
- Клавиша [Enter]: Выбор функции.
- Клавиша [Esc]: Выход из программы.
- Клавиши [y/n]: Подтверждение действий или повтор ввода.

СБОРКА:
Компиляция через MSVC (Visual Studio) или MinGW (g++). 
Рекомендуемая кодировка исходного кода: Windows-1251 (ANSI) 
для корректного отображения кириллицы.
